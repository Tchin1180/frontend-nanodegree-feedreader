# Feed Reader Project Overview

Using [Jasmine](http://jasmine.github.io/) to write test suites and specs to test the Feed Reader App.


## How to use this repo?

1. Clone or download the [Repository](https://github.com/Tchin1180/frontend-nanodegree-feedreader.git) to your local machine.
2. Once the files are on your local machine, simply navigate to the folder where you saved the repo.
3. Open ‘index.html’ in your local browser to view the Feed Reader app and 7 passed tests.
4. The actual test suites and specs are located in the feedreader.js file in this path: '/jasmine/spec/feedreader.js'.
